# mu-server-api-first-sample

Sample app showing how to build and package a [mu-server](https://github.com/3redronin/mu-server) application following the *API First* design with the [openapi-generator-maven-plugin](https://openapi-generator.tech/docs/plugins).

You can run locally by running `App#main` or try building an uber jar.

## Building the uber jar

    mvn package
    java -jar target/mu-server-api-first-sample-1.0-SNAPSHOT.jar
